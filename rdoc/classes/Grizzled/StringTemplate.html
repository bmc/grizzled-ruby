Module Grizzled::StringTemplate
In: lib/grizzled/string/template.rb

StringTemplate - A simple string templating solution modeled after Python‘s StringTemplate library. While Ruby has less need of this kind of library, because of its built-in string interpolation, this module can be useful for interoperating with Python or with the Grizzled Scala library‘s string templating library.
