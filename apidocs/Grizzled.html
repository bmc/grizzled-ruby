    module Grizzled

    Miscellaneous additional Ruby file utility modules and classes.

    Provides a module which, when mixed in, can be used to forward all missing methods to another object.

    Provides a simple stack implementation. See the Grizzled::Stack class for complete details.

    Provides a simple variable substitution capability, similar, in concept, to Python's StringTemplate class. See the Grizzled::String::Template module, the Grizzled::String::UnixShellStringTemplate class and the Grizzled::String::WindowsCmdStringTemplate class for complete details.
