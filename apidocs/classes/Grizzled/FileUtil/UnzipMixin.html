Module Grizzled::FileUtil::UnzipMixin
In: lib/grizzled/fileutil/ziputils.rb

This module provides convenient front-end methods for unzipping files; it uses the ‘rubyzip’ gem under the covers, so you must have ‘rubyzip’ installed to use this class. Mixing this module into your class mixes in methods that will allow you to unzip zip files. Related modules and classes:

A module mixin for zipping zip files.

unzip(zip_file, directory)

Unzips a zip file into a directory.

Parameters:

zip_file
The zip file to unzip.
directory
The directory into which to unzip the file. The directory is created if it doesn‘t already exist.
options
Options hash, as described below.

Options:

Example: 

    import 'grizzled/fileutil/ziputils'
    import 'tmpdir'

    include Grizzled::FileUtil

    Dir.mktmpdir do |d|
      unzip zipfile_path, d
      # muck with unpacked contents
    end
